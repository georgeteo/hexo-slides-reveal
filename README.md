## Customize

- Add these config below to your _config.yml file(which under your blog root directory)

    ```yml
    # hexo-slides-reveal
    slide:
        dir: slides
        separator: ^\n---\n$
        separator_vertical: ^\n--\n$
        charset: utf-8
    ```
